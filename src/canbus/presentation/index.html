Présentation

Le réseau CAN Bus permet de faire communiquer chaque module les uns avec les autres au travers de 2 fils CAN-low CAN-hight. Tous les modules sont donc relié entre eux :

Le bus CAN doit être normalement terminé par des resistantes de 120ohm, mais cela n'est pour le moment pas présent et tout semble fonctionner.

Les informations circulent sous forme de message (sans sens particulier) et chaque module et le PC est en écoute des messages et les traitent s'il leur ai destiné.

Trame d'un message

Module CANable

Le module CANable permet de décoder le bus CAN venant du cockpit via un port USB du PC.

Configuration

Le CANable fonctionne sur les bauds rates suivants:
Serial baud rate 9600
CAN Bus baud rate 125 kbits

Pilote

Pour l’installation des drivers, télécharger ce fichier zip https://github.com/rguilbeau/A320_Documentation/tree/main/_assets/driver (ou depuis le site de l'éditeur https://canable.io/) extraire sont contenu et installer le grâce au clique droit sur le fichier .inf puis "Installer"

Vérifier dans le gestionnaire de périphérique, il devrait être visible en tant que Ports (COM et LPT).

Le COM Port doit indiquer CANtact. Si ce n’est pas le cas, aller sur cette page https://canable.io/updater/ du constructeur et suivre les instructions pour flasher le CANable sur le firmware slcan
