Ajouter une page

  • Copier/coller le contenu du fichier page.html.tpl dans une nouveau fichier new-page.html
    ! Le fichier html doit à la racine du projet
  • Ajouter un item dans le menu ./menu/menu.js
    Les titres des pages et leurs descriptions sont autmatiquement ajouté
  • Dans dossier ./page, créer un dossier du même nom que le fichier html pour y ajouter les assets ./page/new-page/*.js|.css|.img
    ! Il sera par contre nécéssaire de les ajouter manuellement dans le html

Intégrer des diagrammes

  • Créer un diagramme draw.io et l'enregistrer dans le projet au format xml (pas sur le drive)
  • Cliquer sur Fichier > intégrer > HTML Bien vérifier que la case "Inclure une copie de mon diagramme est cochée"
  • Copier / coller ensuite le HTML dans une page du document
