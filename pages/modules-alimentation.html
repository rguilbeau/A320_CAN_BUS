Modules - Alimentation électrique

Vue générale de l'alimentation des modules du Cockpit

Description

Alimentaion générale

Tout le cockpit est alimenté grâce à un transformateur 230 AC / 12V DC (Ampérage à determiner)

Connecteur

Chaque module du cockpit est alimenté par un connecteur qui possède l'alimentation électrique, le bus CAN et la sortie pour intégrer un fusible.

Toutes ces connexions passent ensuite dans le module d'alimentation. Ce module d'alimentation est commun à tous les module du cockpit.
Il permet de :

  • Transferer le 12V au module du cockpit (pour les LED et moteurs)
  • Générer un courant 5V (pour l'électronique)
    MP1584 12v - 5v
  • Transformer le signal CAN bus en signal SPI (sgnal utilisable par un micro-controlleur)
    MCP2515 Can Bus Module TJA1050

Note importante pour l'alimentation des modules du cockpit:

  • Pour un arduino, l'alimentation 5V doit passer par la pin 5V (alimentation direct)
  • Pour un nodemcu, l'alimentation 5V doit passer pa la pin VIN (alimentation via le régulateur de tension du nodemcu (5V vers 3.3V)
  • Tous les autres composants doivent être alimenté par le 5V du module d'alimentation et non du micro-controlleur.
    (Cela permet de ne pas utiliser le régulateur de tension du micro-controlleur, qui a des limites de courant plus faible)
  • Voir : https://riton-duino.blogspot.com/2020/04/arduino-lalimentation.html?m=1
